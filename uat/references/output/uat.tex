  AA: The descriptive statistic of standard deviation can be produced

  AB: The descriptive statistic of variance can be produced based on an

  AC: The descriptive statistic of min can be produced based on an input
  variable

  AD: The descriptive statistic of max can be produced based on an input
  variable

  AE: The descriptive statistic of missing can be produced based on an
  input variable

  AF: Custom descriptive statistics can be produced based on an input
  variable and a specified formula

  AG: Descriptive statistics can be performed across discrete values
  within a grouping variable or a group of grouping variables

  AI: Decimal precision can be specified by the user

  AQ: Shift n counts of values within a variable can be produced

  AS: Dummy values for shift counts can be specified for categories that

  AV: For shift counts when producing n (\%), the denominator can be

  AW: For shift counts when producing n (\%), the denominator can be
  specified using the population data

  AX: For shift counts when producing n (\%), the denominator can be

  AY: Row labels can be manually specified by the user

  BA: Summaries can be stacked on top of one another

  BB: Summaries can be sorted based on manual sorting by presentation

  BD: Summaries can be sorted based on alphabetical sorting based on

  BE: Summaries can be sorted based on a numeric version of the target
  variable if available
\item
  BF: Summary by variables will be sorted by a numeric variable if
  available and then by factor

  BG: Nested layers can be sorted independently using different methods
\item
  BH: Independent layers can be sorted using different methods and
  stacked using common sorting variables

  BI: Count layer default formats can be set at the table level

  BJ: Descriptive statistics layer default formats can be set at the

  BK: Shift layer default formats can be set at the table level

  BM: Option for descriptive statistics layer default formats can be

  BN: Option for shift layer default formats can be specified by the

  BR: Option for setting quantile algorithm choice can be specified by

  BS: Column headers can be added to the output object
\item
  BT: Row breaks can be added between sections based on grouping
  variables

  BU: Row labels can be masked in a hierarchical fashion

  BV: A table object is returned in a format that is ready to be

  BW: Count layers can process a cols argument and separate population

  processing

  data, and a defined subset passed from the table level along with
  normal count layer processing
